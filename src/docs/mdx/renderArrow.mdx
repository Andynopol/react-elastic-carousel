- You can pass a function that returns a react element,   
  This function is a [render prop](https://reactjs.org/docs/render-props.html) and its signature is:  
```
({type, onClick}) => {/* return your element here */}
```

- `type` = `PropTypes.oneOf([consts.PREV, consts.NEXT])`
- `onClick` Is the hook to the inner `onClick` of the `Carousel`. You can attach it wherever you want within the returned component.

## The `consts` object 
` You can import the `consts` object like so:
```
import Carousel, { consts } from 'react-elastic-carousel';
```
