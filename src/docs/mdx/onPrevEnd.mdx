# onPrevEnd
#### `onPrevEnd` will pass the new __current__ `item-object`:  
```
(currentItem) => {/*...*/}
```
- The `item-object` has the shape of:
```
{
    item: {/* your item */},
    index: // the item's index number 
}
```
